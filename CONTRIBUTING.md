# Contributing to claude-brain

Thanks for your interest in contributing! Here's how to get started.

## Ways to Contribute

- **Bug reports** — Open an issue with reproduction steps
- **Feature requests** — Open an issue describing the use case
- **Code** — Fork, branch, PR. Keep changes focused.
- **Docs** — Typo fixes, better examples, translations

## Development Setup

```bash
git clone https://github.com/toroleapinc/claude-brain.git
cd claude-brain
# Run tests
bash tests/run.sh
```

## Pull Request Guidelines

1. One logical change per PR
2. Update docs if behavior changes
3. Add tests for new features when possible
4. Keep commit messages clear

## Code of Conduct

Be kind, be constructive, be helpful. That's it.
